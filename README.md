Proyecto Laboratorio: Manejador de Versiones
Este proyecto consiste en el desarrollo de un sistema de gestión de versiones simplificado, desarrollado en C++ como parte de las entregas finales de la carrera. El objetivo principal es aplicar conceptos avanzados de programación estructurada y manejo de archivos.

Características principales
Control de entregas: Lógica para gestionar versiones de archivos y estados de proyectos.

Modularidad: Separación clara entre archivos de cabecera (.h) e implementación (.cpp).

Compilación: Configurado para ser compilado en entornos Linux/Unix (incluye configuración de ZinjaI).

Cómo ejecutar
Descomprime el archivo o clona el repositorio.

Abre el proyecto en ZinjaI.

Compila y ejecuta el sistema desde la terminal integrada.

Laboratory Project: Version Manager
